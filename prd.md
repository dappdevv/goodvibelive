# Product Requirements Document (PRD)

## Good Vibe - Платформа для мечтателей и творчества

### 1. Обзор продукта

**Название продукта:** Good Vibe  
**Домен:** goodvibe.live  
**Версия:** 1.0.0  
**Дата создания:** 2024

**Описание:** Good Vibe - это современная веб-платформа для творческих людей, предоставляющая доступ к AI-сервисам через токен-экономику и реферальную систему. Платформа объединяет AI-инструменты для генерации контента с криптовалютной экосистемой TAC.

### 2. Целевая аудитория

- **Основная аудитория:** Творческие люди, дизайнеры, контент-мейкеры
- **Вторичная аудитория:** Разработчики, предприниматели, инвесторы в криптовалюту
- **Возраст:** 18-45 лет
- **География:** Глобально (с фокусом на русскоязычные страны)

### 3. Ключевые функции

#### 3.1 Аутентификация

- **Telegram OAuth:** Единственный способ входа в приложение
- **Интеграция с Telegram Mini Apps:** Нативная поддержка Telegram экосистемы

#### 3.2 AI-сервисы

- **AI ChatBot:** Интеллектуальный чат-бот для общения
- **AI Images:** Генерация изображений через Flux и Midjourney
- **AI Videos:** Создание видео через Veo3, RunWay, Sora, Wan, Midjourney
- **AI Music:** Генерация музыки через Suno

#### 3.3 Токен-экономика

- **Внутренняя валюта:** Токены Good Vibe
- **Способы пополнения:**
  - P2P биржа платформы
  - Криптовалютный кошелек TAC
  - Реферальная система

#### 3.4 Реферальная система

- **8 уровней глубины**
- **Максимум 6 рефералов первого уровня**
- **Автоматическое распределение:** 7-й реферал идет на 2-й уровень
- **Вознаграждения:**
  - 1 уровень: 30%
  - 2 уровень: 10%
  - 3 уровень: 5%
  - 4 уровень: 5%
  - 5 уровень: 5%
  - 6 уровень: 5%
  - 7 уровень: 10%
  - 8 уровень: 20%

#### 3.5 Система подарков

- Создание подарков с указанием суммы в токенах
- Привязка к номеру телефона получателя
- Активация через промо-коды
- Автоматическое создание реферальной связи

### 4. Тарифные планы

#### 4.1 FREE

- **Стоимость:** Бесплатно
- **Функции:** Базовый доступ к AI-сервисам

#### 4.2 SMART

- **Стоимость:** 1000 токенов/месяц
- **Скидка:** -30% на AI-сервисы

#### 4.3 PRO

- **Стоимость:** 4000 токенов/месяц
- **Скидка:** -50% на AI-сервисы

#### 4.4 DREAM

- **Стоимость:** 10000 токенов/месяц
- **Скидка:** -60% на AI-сервисы

### 5. Административная панель

#### 5.1 Управление планами

- Создание и редактирование тарифных планов
- Настройка цен и скидок
- Управление функциями планов

#### 5.2 Конструктор промо-акций

- Создание рекламных кампаний
- Настройка промо-кодов
- Аналитика эффективности

#### 5.3 Управление пользователями

- Поиск и просмотр пользователей
- Модерация контента

#### 5.4 Мониторинг системы

- Просмотр логов приложения
- Отслеживание ошибок
- Аналитика производительности

#### 5.5 Поддержка пользователей

- Встроенный чат поддержки
- Система тикетов
- База знаний

### 6. Криптовалютная интеграция

#### 6.1 TAC кошелек

- Создание кошелька в сети TAC
- Связывание с Telegram кошельком
- Поддержка множественных токенов

#### 6.2 Поддерживаемые токены

- Список токенов настраивается в админ-панели
- Автоматический курс обмена
- Безопасные транзакции

### 7. Технические требования

#### 7.1 Фронтенд

- **Framework:** React, Next.js
- **UI библиотеки:** Radix UI
- **Стилизация:** Tailwind CSS
- **Состояние:** Zustand
- **Запросы:** TanStack React Query
- **Формы:** React Hook Form, Zod
- **Интернационализация:** i18next

#### 7.2 Бэкенд

- **База данных:** Supabase
- **Аутентификация:** Supabase Auth + Telegram
- **API:** REST/GraphQL через Supabase

#### 7.3 Инфраструктура

- **Монorepo:** Turbo
- **Деплой:** Vercel

#### 7.4 Криптовалютная интеграция

- **TON Connect UI:** Для интеграции с TON
- **Wagmi:** Для работы с TAC
- **TAC SDK:** Для взаимодействия с сетью TAC

### 8. Безопасность

#### 8.1 Аутентификация

- Telegram OAuth
- JWT токены через Supabase
- Двухфакторная аутентификация (опционально)

#### 8.2 Криптобезопасность

- Безопасное хранение приватных ключей
- Подписание транзакций на клиенте
- Валидация транзакций

#### 8.3 Защита данных

- Шифрование персональных данных
- GDPR compliance
- Регулярные аудиты безопасности

### 9. Производительность

#### 9.1 Целевые метрики

- **Время загрузки:** < 3 секунд
- **Время отклика AI-сервисов:** < 30 секунд
- **Доступность:** 99.9%

#### 9.2 Масштабируемость

- Горизонтальное масштабирование
- CDN для статических ресурсов
- Кэширование AI-результатов

### 10. Аналитика и метрики

#### 10.1 Пользовательские метрики

- Количество активных пользователей
- Конверсия в платные планы
- Удержание пользователей

#### 10.2 Бизнес-метрики

- Объем транзакций в токенах
- Доход от подписок
- Эффективность реферальной системы

#### 10.3 Технические метрики

- Производительность AI-сервисов
- Время отклика системы
- Количество ошибок

### 11. Ограничения и исключения

#### 11.1 Не включается в MVP

- Системы оплаты дебетовыми картами
- Банковские переводы
- Другие способы оплаты кроме криптовалют

#### 11.2 Ограничения

- Только Telegram аутентификация
- Фокус на TAC экосистеме
- Ограниченная география (первоначально)

### 12. План развития

#### 12.1 Фаза 1 (MVP)

- Базовая аутентификация через Telegram
- Основные AI-сервисы
- Простая токен-экономика
- Базовая реферальная система

#### 12.2 Фаза 2

- Расширенные AI-инструменты
- Улучшенная админ-панель

#### 12.3 Фаза 3

- Интеграция с другими блокчейнами
- Расширенная аналитика
- API для разработчиков

### 13. Риски и митигация

#### 13.1 Технические риски

- **Риск:** Нестабильность AI-сервисов
- **Митигация:** Резервные провайдеры, мониторинг

#### 13.2 Бизнес-риски

- **Риск:** Изменения в криптовалютном регулировании
- **Митигация:** Юридическая экспертиза, адаптивность

#### 13.3 Рыночные риски

- **Риск:** Конкуренция со стороны крупных игроков
- **Митигация:** Уникальное позиционирование, качество продукта

### 14. Успешные критерии

#### 14.1 Количественные

- 10,000 активных пользователей в первый год
- $100,000 месячный оборот через 18 месяцев
- 95% доступность системы

#### 14.2 Качественные

- Высокий пользовательский опыт
- Стабильная работа AI-сервисов
- Позитивные отзывы сообщества
